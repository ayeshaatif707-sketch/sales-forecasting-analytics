# 📊 Sales Forecasting Analytics Project

## 📌 Overview
This project is an end-to-end **Sales Forecasting & Data Analytics solution** built using Python.  
It covers data generation, cleaning, exploratory data analysis (EDA), time-series forecasting, model evaluation, and automated Excel reporting.

The goal is to simulate real-world business sales data and build predictive models to forecast future sales trends.

---

## 🚀 Key Features

- 📦 Synthetic real-world sales dataset (3000+ rows)
- 🧹 Data cleaning & preprocessing
- 📊 Exploratory Data Analysis (EDA)
- 📈 Time series aggregation (monthly sales trends)
- 🤖 Forecasting models:
  - ARIMA
  - Random Forest Regression
- 📉 Model evaluation (RMSE, MAE)
- 📊 Visual insights (trend & category analysis)
- 📁 Automated Excel report generation

---

## 🛠️ Tech Stack

- Python 🐍
- Pandas & NumPy
- Matplotlib & Seaborn
- Scikit-learn
- Statsmodels (ARIMA)
- OpenPyXL (Excel reporting)
- Jupyter Notebook
